Zetachain Testnet Guide

In this article, I will go through Zetachain Testnet, learn how to use it, and gain Zeta points.
Before That, Let’s examine what Zetachain is and why we want to participate in their testnet.

What is Zetachain?

ZetaChain is the foundational, public blockchain that enables omnichain, generic smart contracts and messaging between any blockchain. It solves the problems of “cross-chain” and “multi-chain” and aims to extend the crypto and global financial ecosystem to anyone. ZetaChain envisions and supports a truly fluid, multi-chain crypto ecosystem where users and developers can move between and appreciate the benefits of any blockchain: payments, DeFi, liquidity, games, art, social graphs, performance, security, privacy, and so on.

  • Unlike the more common two-way pegging, there is no wrapping and, therefore, no multiple representations of the same underlying asset.
  • The only (native) value that can cross-chain is via the ZETA coin, which substantially reduces the attack surface, resulting in an easier-to-understand audit and, therefore, higher security. For example, we can check the total supply at the contract mint site.
  • Users can pay ZETA for the cross-chain service that ZetaChain provides and gas on the destination chain in a single step/bundle.

Source: zetachain.com

Zetachain Testnet

  • If you are a new user of web3, You should know that you need a wallet to connect platforms. In my opinion, Metamask is the best wallet that you can use. Metamask has a Chrome and Mozilla extension; you can download it from their official sources.

    If you already have Metamask, Make sure you are using the Testnet wallet, or you can add a new address to your Metamask.
  • The next step is to get some testnet tokens. You can get testnet tokens on the below websites:
    goerlifaucet.com
    faucet.polygon.technology
    testnet.binance.org/faucet-smart
  • Now, You can use my invitation code and verify your Twitter and wallet for the Zetachain testnet. All you need is to click on the below link ( You will get 5K Zeta point by joining the testnet):
    labs.zetachain.com/leaderboard
  • The Only action you can do for the current Testnet is to use the Swap feature of Zetachain. Before using, the cross-chain swap, get some Testnet Zeta tokens from the ‘Get Zeta’ menu.
  • This Section is easy. If you verify your Twitter and your wallet, You will get Zeta tokens with one click. ( Remember, These are only testnet tokens and don’t have value.)
  • You can go to the Swap tab if you have Testnet tokens and have verified your Twitter and wallet.
  • If you never use a swap, You can follow the below instruction:
    1. Select the Network that you have testnet tokens
    2. Select the Testnet token
    3. Select Destination Network
    4. Select the token that you want to get in the destination chain
    5. Enter The amount of swap

    You can swap any tokens from and to any chain.

    After you set your swap, Click on Review order. The next step should be to give Zetachain permission to access the requested token.

    Click on Allow Zeta Transfer. After that, You will see the Metamask pop-up. In this pop-up, you can check permissions, Transfer fees, and other valuable data. By confirming the pop-up, you give access to Zetachain.

    You have almost done. After that, Click on the Swap button. Again, You will see the Metamask pop-up, which should confirm the next step.

  • Congratulation. You did it. All you need is to wait a minute for your swap to complete.
